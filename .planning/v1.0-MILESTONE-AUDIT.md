## Cross-Phase Integration

**Connected:** 47 key exports properly wired across phases
**Missing:** 4 critical connections

### Critical Integration Gaps

1. **Metadata bucket mismatch** (HIGH) — `create.go` writes to `cfg.StateBucket` (empty by default, gated on `KM_STATE_BUCKET`); `list.go`/`status.go` read from hardcoded `"tf-km-state"`. No shared config ensures these match. Affects PROV-03, PROV-04, CFUI-03.

2. **TTL auto-destroy not wired** (HIGH) — EventBridge fires TTL Lambda which uploads artifacts and sends notification. The Lambda comment says "delegated to a separate cleanup job" but no such job exists. Sandbox resources persist indefinitely after TTL. Affects PROV-05.

3. **Idle auto-destroy not wired** (HIGH) — `IdleDetector.OnIdle` cancels the audit-log sidecar's context, exiting the process. No `km destroy` or `terragrunt.Destroy()` is triggered. EC2 instances keep running; ECS behavior is non-deterministic. Affects PROV-06.

4. **ECS budget top-up missing** (MEDIUM) — `km budget add` resumes EC2 via `StartInstances` but has no ECS path. ECS sandboxes suspended by budget enforcer cannot be resumed. Affects BUDG-08.
