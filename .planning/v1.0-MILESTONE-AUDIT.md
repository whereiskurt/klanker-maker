## Cross-Phase Integration

### Critical Integration Gaps

**1. Sidecar Deployment Pipeline (NETW-02, NETW-03, OBSV-01, OBSV-02)**
No Makefile, `km build`, or operator workflow exists to compile Go sidecar binaries and upload to S3 (EC2) or push to ECR (ECS). Without this, network policy enforcement and audit logging are inactive on all live sandboxes.

**2. Missing Live Infrastructure (PROV-05, BUDG-02 through BUDG-09, MAIL-01)**
Four Terraform modules (`ttl-handler`, `dynamodb-budget`, `ses`, `budget-enforcer`) have no `infra/live/use1/` Terragrunt configs. TTL lifecycle, budget enforcement, and email all require manual operator deployment with no documented procedure.

**3. ECS Container Image Resolution (PROV-10, NETW-02, NETW-03)**
Generated ECS service.hcl contains literal `${var.dns_proxy_image}` strings that don't resolve to ECR URIs. ECS substrate sandboxes will fail to start sidecar containers.
