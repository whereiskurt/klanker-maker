<objective>
Add the compile-time mention-only resolver to `pkg/compiler/userdata.go` and emit the
`KM_SLACK_MENTION_ONLY` env var into the existing `notifyEnv` map (the same map that emits
`KM_NOTIFY_SLACK_ENABLED`, `KM_SLACK_CHANNEL_ID`, `KM_AGENT`, etc.). POL-04 + POL-11.

Purpose: The compiler is the single source of truth that takes a profile + channel mode and
produces the effective `mention_only` boolean. This plan locks that resolution in code so the
bridge (Plan 03) can consume the value via a single env var without re-deriving it from the profile.

Note on architecture (recorded for the executor): RESEARCH.md § Pattern 4 and § Q1/Q5 discuss
two viable architectures — per-sandbox DDB attribute vs install-level Lambda env var. CONTEXT.md
locks `KM_SLACK_MENTION_ONLY` as the env var name and "emitted into the bridge config from the
resolved profile field." This plan implements the install-level env var path: the compiler
writes the value into the sandbox's notify env (so the value is reachable for diagnostics and
for any future sandbox-side reader), and Plan 03 separately wires the same env var into the
bridge Lambda's Terraform variables. Per-sandbox DDB-attribute routing is explicitly deferred
(see RESEARCH.md Pitfall 2 note + Open Question #1).

Output: One helper function + ~10 lines of emission code in userdata.go + 9-case live test table.
</objective>

    ```go
    // resolveMentionOnly returns the effective mention-only bool for the bridge,
    // given a CLI spec. Phase 91.
    //
    // Resolution order:
    //   1. Explicit override (cli.NotifySlackInboundMentionOnly != nil) wins.
    //   2. Mode 2 (per-sandbox, cli.NotifySlackPerSandbox == true) defaults to false (every-message).
    //   3. Mode 1 (shared) and Mode 3 (cli.NotifySlackChannelOverride != "") default to true (mention-only).
    //
    // Returns false when cli is nil — defensive: caller already gates on NotifySlackEnabled,
    // so this path should not be hit, but a nil deref would crash the compiler.
    func resolveMentionOnly(cli *profile.CLISpec) bool {
        if cli == nil {
            return false
        }
        if cli.NotifySlackInboundMentionOnly != nil {
            return *cli.NotifySlackInboundMentionOnly
        }
        if cli.NotifySlackPerSandbox {
            return false
        }
        return true
    }
    ```
