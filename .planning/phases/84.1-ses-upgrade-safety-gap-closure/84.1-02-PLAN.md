<objective>
Close GAP-4 and GAP-5 from Phase 84 UAT: a wedged `terragrunt apply` produced no stderr (because the process never returned), `runner.Apply` had no upper bound, and `km init` blocked indefinitely (10+ minutes in UAT — see `84-10-UAT.md` lines 53-72).

Per UAT observations #4 + #5 (`84-10-UAT.md` lines 151-153):
- "The Verbose=false runner captures stderr and only prints on failure exit, but the wedged-terraform scenario produces no stderr because the process never returned. Operator-visible state: indefinite `Applying ses...` with no progress indicator. Lasted 10+ minutes before manual Ctrl-C."
- "`runner.Apply` has no upper bound. Combined with Gap 4, a wedged terraform child process blocks `km init` indefinitely. Future similar conditions should surface as a timeout-with-context error."

Purpose: Make `km init` fail fast and visibly when a regional module wedges, instead of silently hanging. Operator should see either (a) live progress (when --verbose) or (b) a periodic heartbeat (default), and a bounded timeout error if any module exceeds its limit.

Output: Bounded runner with progress feedback + per-module timeout in RunInitWithRunner.
</objective>

    ```go
    func (r *Runner) runCommand(cmd *exec.Cmd) error {
        ctx := cmdContext(cmd) // helper: returns the ctx passed to exec.CommandContext
        // ... existing Stdout/Stderr wiring (Verbose vs quiet) ...

        if err := cmd.Start(); err != nil {
            return err
        }

        // Quiet-mode heartbeat goroutine
        done := make(chan struct{})
        if !r.Verbose && r.ProgressInterval > 0 {
            go r.runHeartbeat(ctx, done, cmd)
        }

        waitErr := cmd.Wait()
        close(done)

        // Surface ctx.Err() over cmd.Wait()'s "killed" error when ctx caused the kill.
        if ctx.Err() == context.DeadlineExceeded {
            return fmt.Errorf("terragrunt apply: %w (process killed after timeout)", ctx.Err())
        }
        if ctx.Err() == context.Canceled {
            return fmt.Errorf("terragrunt apply: %w", ctx.Err())
        }
        return waitErr
    }

    func (r *Runner) runHeartbeat(ctx context.Context, done <-chan struct{}, cmd *exec.Cmd) {
        start := time.Now()
        ticker := time.NewTicker(r.ProgressInterval)
        defer ticker.Stop()
        for {
            select {
            case <-done:
                return
            case <-ctx.Done():
                return
            case t := <-ticker.C:
                fmt.Fprintf(os.Stderr, "  ... still running (%s elapsed, pid=%d)\n",
                    t.Sub(start).Round(time.Second), cmd.Process.Pid)
            }
        }
    }
    ```
