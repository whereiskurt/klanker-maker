## Accomplishments
- Added `SlackInboundDLQName` / `GitHubInboundDLQName` / `DLQArn` deterministic helpers (no SQS API call) and `GetQueueARN` (one `GetQueueAttributes` call for `QueueArn`).

## Decisions Made
- Shared DLQ must be FIFO to be attachable to FIFO source queues (RESEARCH Pitfall 1).
- DLQ mirrors 14-day retention for inspection time; omits `VisibilityTimeout` (no meaning for a terminal DLQ).
