## Decisions Made
- Used `makeEmptyRepoRoot` + `KM_REPO_ROOT` env var override to make `runInitPlanWithWriter` fast without real terragrunt invocations — a fake repo root with no modules causes `RunInitPlanWithRunner` to skip all modules and return nil in <1s
- Added `isolateExportEnvVars` helper to `config_load_drift_test.go` to pre-register all `ExportTerragruntEnvVars` output env vars via `t.Setenv` — without this, `os.Setenv` calls inside the function leak `KM_REGION_LABEL` and similar vars across tests, causing false failures in negative tests
- Placed `TestRunBootstrap_DriftWarn_KM_REGION` behind `t.Skip` because it requires `cfg.YAMLDefaults` (Plan 07 field) — compiling without it would require forward-declaring a struct field, which Plan 07 must own
