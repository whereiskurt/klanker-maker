# Phase 84.3 Plan 09: Placeholder Bucket Validation Summary

**One-liner:** Wire `isPlaceholderBucket` into `config.Load()` and `validateArtifactsBucket` into `runInitPlanWithWriter` to hard-fail on placeholder `artifacts_bucket` values before any AWS or terragrunt work.

## What Was Built

### Gap 3 closure (config.Load())

Added `isPlaceholderBucket` package-private helper to `internal/app/config/config.go`. It detects:
- Angle-bracket tokens: any string containing `<...>`
- The literal example sentinel: `km-artifacts-12345`
- Returns `false` for empty string (fresh install; km configure will derive the correct value)

Wired into `Load()` just before `return cfg, nil` (after the `YAMLDefaults` snapshot block from Plan 07):

```go
if isPlaceholderBucket(cfg.ArtifactsBucket) {
    return nil, fmt.Errorf(
        "artifacts_bucket=%q is a placeholder; re-run `km configure` to derive ${prefix}-artifacts-${account_id} automatically",
        cfg.ArtifactsBucket,
    )
}
```

**isPlaceholderBucket location:** `internal/app/config/config.go` lines 451–469 (after the final `return cfg, nil`).

**Insertion in Load():** line 441 (before `return cfg, nil` at line 448).

### Gap 4 closure (runInitPlan / runInitPlanWithWriter)

Added `validateArtifactsBucket` call at the top of both:
- `runInitPlan` (line 1131 in init.go) — production entry point, before AWS credential check
- `runInitPlanWithWriter` (line 1171 in init.go) — test seam used by `TestInitPlan_HardFailsOnPlaceholderBucket`

Both wrap with: `return fmt.Errorf("artifacts_bucket misconfigured: %w", err)`

### configure.go wizard immunity

`grep -c "config\.Load(" internal/app/cmd/configure.go` returns **0**.

`runConfigure` reads existing km-config.yaml via `os.ReadFile + yaml.Unmarshal` directly — it does NOT call `config.Load()`. Therefore the new `config.Load()` validation does NOT block the wizard from running even when the existing km-config.yaml contains a placeholder bucket. The wizard is the cure for a placeholder bucket and must not be blocked by the same placeholder it exists to fix.
