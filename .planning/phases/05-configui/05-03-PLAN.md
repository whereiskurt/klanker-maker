1. Create `cmd/configui/handlers_secrets.go`:

Define narrow SSMAPI interface following project convention:
```go
type SSMAPI interface {
    GetParametersByPath(ctx context.Context, input *ssm.GetParametersByPathInput, optFns ...func(*ssm.Options)) (*ssm.GetParametersByPathOutput, error)
    GetParameter(ctx context.Context, input *ssm.GetParameterInput, optFns ...func(*ssm.Options)) (*ssm.GetParameterOutput, error)
    PutParameter(ctx context.Context, input *ssm.PutParameterInput, optFns ...func(*ssm.Options)) (*ssm.PutParameterOutput, error)
    DeleteParameter(ctx context.Context, input *ssm.DeleteParameterInput, optFns ...func(*ssm.Options)) (*ssm.DeleteParameterOutput, error)
}
```

Add `ssmClient SSMAPI` field to Handler struct (update handlers.go if needed).

Implement handlers:

- `handleSecretsPage(w, r)` -- renders secrets.html full page. Calls listKMSecrets() to populate initial table.

- `handleSecretsList(w, r)` -- GET /api/secrets. Uses `ssm.NewGetParametersByPathPaginator` with path `/km/`, `Recursive: true`, `WithDecryption: false` (metadata only). Returns JSON array of `{name, type, lastModified, version}`. If HTMX request, render secret_row partials instead.

- `handleSecretDecrypt(w, r)` -- GET /api/secrets/{name...}. Validate name starts with `/km/`. Call `ssm.GetParameter` with `WithDecryption: true`. Return JSON `{name, value, type}`. This is the only endpoint that returns decrypted values -- called on "reveal" click.

- `handleSecretPut(w, r)` -- PUT /api/secrets/{name...}. Validate name starts with `/km/`. Read body as the secret value string. Call `ssm.PutParameter` with `Type: SecureString`, `Overwrite: true`. Auto-detect KMS key from `.sops.yaml` if present (read `creation_rules[0].kms` key ARN), otherwise use AWS default SSM encryption. Return 200 on success.

- `handleSecretDelete(w, r)` -- DELETE /api/secrets/{name...}. Validate name starts with `/km/`. Call `ssm.DeleteParameter`. Return 200. Swallow ParameterNotFound error (idempotent delete).

Helper: `validateKMPrefix(name string) error` -- returns error if name doesn't start with `/km/`.
