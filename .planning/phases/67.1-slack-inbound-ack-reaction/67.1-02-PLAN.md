<objective>
Operator-facing scope verification: when the Slack App is missing the `reactions:write` Bot Token Scope, both `km slack init` and `km doctor` must surface a clear actionable warning. Without this, operators who deploy the bridge code from Plan 01 but forget to add the scope in the Slack App config will see silent CloudWatch warnings (`events: reaction failed` with `missing_scope`) instead of an upfront diagnostic.

This plan extends two existing scope-check functions:
1. `VerifyEventsAPIScopes` in `internal/app/cmd/slack.go` (called by `km slack init`)
2. `checkSlackAppEventsScopes` in `internal/app/cmd/doctor_slack.go` (called by `km doctor`)

Both functions already declare `required := []string{"channels:history", "groups:history"}` and check it against the bot's actual scope list. The change is mechanically trivial — append `"reactions:write"` to each slice, update the success-message string, extend tests.

Purpose: This plan is **fully parallelizable with Plan 01** because it touches a disjoint set of files (`internal/app/cmd/*` vs `pkg/slack/bridge/*`). No code interface dependency between the two plans. Both can execute in Wave 1.

Output: Updated scope-check functions + updated tests + new test cases proving `reactions:write` is checked. Operator gets a clear FAIL/warning when the scope is missing.

Note: The scope is checked but not added by automation — adding the scope requires operator action in the Slack App admin UI. That manual step is documented in Plan 03 (operator runbook + UAT).
</objective>

// VerifyEventsAPIScopes checks whether the provided scope list contains the
// scopes required for Slack Events API inbound (channels:history, groups:history).
// Returns (allPresent bool, missing []string).
// Exported for testability.
func VerifyEventsAPIScopes(scopes []string) (bool, []string) {
    required := []string{"channels:history", "groups:history"}
    scopeSet := make(map[string]bool, len(scopes))
    for _, s := range scopes {
        scopeSet[s] = true
    }
    var missing []string
    for _, r := range required {
        if !scopeSet[r] {
            missing = append(missing, r)
        }
    }
    return len(missing) == 0, missing
}
```

func checkSlackAppEventsScopes(
    ctx context.Context,
    getScopes func(context.Context) ([]string, error),
) CheckResult {
    name := "Slack App events scopes"
    if getScopes == nil {
        return CheckResult{Name: name, Status: CheckSkipped, Message: "Slack auth-test scopes func not configured"}
    }
    scopes, err := getScopes(ctx)
    if err != nil {
        return CheckResult{Name: name, Status: CheckWarn, Message: fmt.Sprintf("could not check Slack scopes: %v", err)}
    }

    required := []string{"channels:history", "groups:history"}
    scopeSet := make(map[string]bool, len(scopes))
    for _, s := range scopes {
        scopeSet[s] = true
    }
    var missing []string
    for _, r := range required {
        if !scopeSet[r] {
            missing = append(missing, r)
        }
    }
    if len(missing) > 0 {
        return CheckResult{
            Name:        name,
            Status:      CheckError,
            Message:     fmt.Sprintf("Slack App missing required scopes for inbound: %s", strings.Join(missing, ", ")),
            Remediation: "Add scopes via Slack App config → OAuth & Permissions → Bot Token Scopes, then run 'km slack rotate-token' to refresh the SSM-cached token",
        }
    }
    return CheckResult{
        Name:    name,
        Status:  CheckOK,
        Message: "Slack App has all required inbound scopes (channels:history, groups:history)",
    }
}
```
