    1. **Define ApprovalResult type:**
       ```go
       type ApprovalResult struct {
           Found    bool   // true if a reply matching the action was found
           Approved bool   // true if reply contains APPROVED
           Denied   bool   // true if reply contains DENIED
           Reply    string // the reply body text
       }
       ```

    2. **Create PollForApproval function:**
       ```go
       func PollForApproval(ctx context.Context, client MailboxS3API, bucket, sandboxID, emailDomain, action string) (*ApprovalResult, error)
       ```

       Implementation:
       - Call `ListMailboxMessages(ctx, client, bucket, sandboxID, emailDomain)` to get all messages
       - For each message key, call `ReadMessage(ctx, client, bucket, key)` to get raw MIME
       - Parse with `mail.ReadMessage` (lightweight parse, no signature verification needed for operator replies — operator is external, not a sandbox)
       - Match by subject: look for subjects containing `"Re:"` and the `action` string (case-insensitive)
       - If found, scan body for "APPROVED" or "DENIED" (case-insensitive using `strings.Contains` + `strings.ToUpper`)
       - Return `ApprovalResult` with findings
       - If no matching reply found, return `&ApprovalResult{Found: false}`

       - TestPollForApproval_NotFound: no matching reply in mailbox
       - TestPollForApproval_CaseInsensitive: "approved" (lowercase) still matches
