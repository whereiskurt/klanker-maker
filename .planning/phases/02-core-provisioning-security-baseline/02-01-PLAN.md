<objective>
Build the profile compiler package that translates a SandboxProfile struct into Terragrunt artifacts (service.hcl for both EC2 and ECS substrates, user-data.sh for EC2, SG egress rules, IAM session policy, and secrets injection config). Also extend the ec2spot Terraform module to accept and apply SG egress rules and IAM session policy from the compiler output.

Purpose: The compiler is the bridge between the declarative profile YAML and the Terraform infrastructure. It is a pure function with no AWS side effects -- fully testable without credentials. The SG egress rules and IAM session policy MUST flow through the generated service.hcl into the Terraform module inputs so they actually reach AWS infrastructure. Without this, NETW-01 (SG egress enforcement) and NETW-04 (IAM session policy) would silently not be applied.
Output: `pkg/compiler/` package with Compile() function, tests, and all generation helpers. Updated ec2spot Terraform module with sg_egress_rules and iam_session_policy variables.
</objective>
