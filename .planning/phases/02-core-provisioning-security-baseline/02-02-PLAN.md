    - TestRunnerOutputCommand: Runner.Output() runs terragrunt output -json and returns parsed JSON

  <action>
    Create `pkg/terragrunt/` package.

    **runner.go:** Define Runner struct and execution methods:
    ```go
    type Runner struct {
        AWSProfile string // e.g. "klanker-terraform"
        RepoRoot   string // absolute path to repo root (for CLAUDE.md anchor)
    }
    func NewRunner(awsProfile, repoRoot string) *Runner
    func (r *Runner) Apply(ctx context.Context, sandboxDir string) error
    func (r *Runner) Destroy(ctx context.Context, sandboxDir string) error
    func (r *Runner) Output(ctx context.Context, sandboxDir string) (map[string]interface{}, error)
    ```
    - Apply/Destroy use `os/exec.CommandContext` with `cmd.Stdout = os.Stdout` and `cmd.Stderr = os.Stderr` for real-time streaming (per locked decision: stream Terraform output in real time)
    - Set `cmd.Dir = sandboxDir`
    - Set env: `AWS_PROFILE=<r.AWSProfile>`
    - Apply uses `-auto-approve` flag
    - Output runs `terragrunt output -json`, captures stdout into buffer, parses as JSON map

    **sandbox.go:** Sandbox directory management:
    ```go
    func CreateSandboxDir(repoRoot, sandboxID string) (string, error)
    func PopulateSandboxDir(sandboxDir, serviceHCL, userData string) error
    func CleanupSandboxDir(sandboxDir string) error
    ```
    - CreateSandboxDir: copies `infra/live/sandboxes/_template/terragrunt.hcl` to `infra/live/sandboxes/<sandboxID>/terragrunt.hcl`. Returns the absolute path to the new directory.
    - PopulateSandboxDir: writes `service.hcl` (always) and `user-data.sh` (if non-empty) into the sandbox directory.
    - CleanupSandboxDir: `os.RemoveAll(sandboxDir)`.

    **Tests:** Unit tests using temp directories. For Runner tests, do NOT actually run terragrunt -- test that the command is constructed correctly by inspecting cmd.Args, cmd.Dir, cmd.Env. Use a test helper that creates a mock sandbox dir with template files.
  </action>

  </verify>

    - Sandbox dir creation copies template terragrunt.hcl correctly
    - Sandbox dir population writes service.hcl and optional user-data.sh
    - Cleanup removes sandbox directory
    - All tests pass without requiring terragrunt binary
  </done>

    ```go
    type SandboxLocation struct {
        SandboxID     string
        S3StatePath   string // "tf-km/sandboxes/<sandbox-id>"
        ResourceCount int
        ResourceARNs  []string
    }

    // TagAPI interface for testability (mock the AWS API)

        GetResources(ctx context.Context, params *resourcegroupstaggingapi.GetResourcesInput, ...) (*resourcegroupstaggingapi.GetResourcesOutput, error)
    }

    func FindSandboxByID(ctx context.Context, client TagAPI, sandboxID string) (*SandboxLocation, error)
    ```

    type EC2API interface {
