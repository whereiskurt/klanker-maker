## Decisions Made

- The `moved {}` block in v2.0.0/main.tf is preserved for documentation and future-proofing even though AWS SSM doesn't support document rename (terraform falls back to destroy+create). This matches RESEARCH.md Pitfall 3 documentation.
- `TestModuleV1HardcodedLiteralsWarn` is WARN-only with no regression gate: v1.0.0's `name = var.document_name` is parametric (not a literal), so the resource-block audit cannot catch it. The test provides discovery for other v1.0.0 modules (ecs, efs, s3-replication, ses, scp all surface warnings).
- `execSSMSession` signature extended with `cfg *config.Config` (rather than pre-computing docName at the call site) to keep the function self-contained and enable future cfg-based session configuration.
