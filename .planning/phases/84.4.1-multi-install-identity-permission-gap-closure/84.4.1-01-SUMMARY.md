## Decisions Made

- **In-place v2.0.0 update**: scp/v2.0.0 was shipped 2026-05-17 and has a single deployer (canonical km install). In-place change causes ~6 in-place property updates on `aws_organizations_policy.sandbox_containment`, zero destroys, sub-10s window. No version bump required.
- **Account wildcard accepted**: `arn:aws:iam::*:role/*-suffix` trusts roles from any account. Primary guard is operator-only `IAM:CreateRole` in the application account; SCP is defense-in-depth. Security trade-off documented in OPERATOR-GUIDE.md by Wave 4 plan 84.4.1-06.
- **Signature backward compat**: `BuildSCPPolicyFromPrefix(resourcePrefix, applicationAccountID, allowedRegion)` signature unchanged so `runBootstrap` caller at line 1374 requires zero edits. `_ = resourcePrefix` and `_ = applicationAccountID` satisfy Go's unused-variable compiler check.
