Create `pkg/aws/rotation.go` with:

1. **Narrow interfaces** following doctor.go/identity.go pattern:
   - `RotationSSMAPI` — PutParameter, GetParameter, GetParametersByPath (reuse IdentitySSMAPI where possible, extend with GetParametersByPath)
   - `RotationS3API` — PutObject for CA cert upload
   - `RotationDynamoAPI` — reuse IdentityTableAPI
   - `RotationCWAPI` — reuse CWLogsAPI from cloudwatch.go

2. **RotationAuditEvent struct** — Event, SandboxID, KeyType, BeforeFP, AfterFP, Timestamp, Success, Error fields (JSON-tagged)

3. **ed25519Fingerprint(pub ed25519.PublicKey) string** — sha256 hash of public key bytes, first 8 bytes as hex: `sha256:XXXXXXXXXXXXXXXX`

4. **ecdsaFingerprint(cert []byte) string** — sha256 hash of DER-encoded cert, first 8 bytes as hex

5. **UpdateIdentityPublicKey(ctx, dynamoClient, tableName, sandboxID, pubKey, encPubKey)** — unconditional PutItem that overwrites the DynamoDB row. CRITICAL: Do NOT use PublishIdentity (it has attribute_not_exists). Copy the item building logic from PublishIdentity but omit the ConditionExpression. Preserve existing alias, allowedSenders, and policy fields by reading current record first with FetchPublicKey and merging.

6. **RotateSandboxIdentity(ctx, ssmClient, dynamoClient, sandboxID, kmsKeyID, tableName) (oldFP, newFP string, err error)** — Fetch old public key via FetchPublicKey (nil = fresh sandbox), generate new via GenerateSandboxIdentity, update DynamoDB via UpdateIdentityPublicKey, return fingerprints.

7. **RotateProxyCACert(ctx, s3Client, bucket string) (oldFP, newFP string, err error)** — Extract cert generation logic from init.go's ensureProxyCACert into a reusable function. Generate ECDSA P-256 key, create self-signed x509 CA cert (5-year validity), PEM-encode, upload to S3 at `sidecars/km-proxy-ca.crt` and `sidecars/km-proxy-ca.key`. Get old cert from S3 first for fingerprint (ignore error if not found). Return fingerprints.

8. **ReEncryptSSMParameters(ctx, ssmClient, sandboxID, kmsKeyID string) (int, error)** — GetParametersByPath for `/sandbox/{sandboxID}/` with Recursive=true, WithDecryption=true. For each parameter, PutParameter with Overwrite=true, Type=SecureString, KeyId=kmsKeyID. Return count of re-encrypted params.

9. **WriteRotationAudit(ctx, cwClient, event RotationAuditEvent) error** — Create log group `/km/credential-rotation` (idempotent via EnsureLogGroup), create log stream named `{date}/{event.Event}`, write JSON-marshaled event via PutLogEvents.

All functions follow existing error handling patterns (return errors, don't panic). Use `aws.String()` helpers consistently.
