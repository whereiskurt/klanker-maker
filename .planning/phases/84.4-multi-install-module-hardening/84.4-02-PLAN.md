<objective>
Create `infra/modules/scp/v2.0.0/` — the per-install SCP module — with every hardcoded
`km-` literal replaced by `${var.resource_prefix}-`, a 5,120-byte plan-time guard, and
companion Go unit tests covering the 4KB safety threshold.

This is the load-bearing fix for Phase 84.4: today the SCP policy *content*
(trusted_arns_*) actively DENIES a second install's `${prefix}-*` lambdas at runtime —
even after manual import — so the second install's sandboxes can't actually run. v2.0.0
templates the trusted_arns_* locals from `var.resource_prefix` so each install's SCP
guards its OWN install's lambda role names.

This plan also updates `BuildSCPPolicy` in `internal/app/cmd/bootstrap.go` (used by
`runShowPrereqs` for display) to accept `resourcePrefix string` so the prereq display
matches reality for non-`km` prefixes.

Wave 1 plans (01/02/03/04) are parallel-safe — `files_modified` lists are disjoint
across modules. The audit test (Plan 01) will pick up this v2.0.0 directory after
both plans land.

Purpose: SCP-PREFIX-NAMESPACING — the highest-impact requirement in Phase 84.4.
Without it, multi-install runtime is broken.

Output: scp/v2.0.0/ module + bootstrap.go function-signature update + tests.
Backward compat: with `resource_prefix=km` (default), rendered output is byte-identical
to v1.0.0 — verified by zero-diff `km init --plan` against existing `km` install (Plan 05).
</objective>

    ```go
    package cmd

    import (
        "strings"
        "testing"
    )

    func TestBuildSCPPolicySize(t *testing.T) {
        cases := []struct {
            prefix     string
            maxBytes   int
            wantNoKM   bool  // for non-km prefixes, assert "km-" doesn't appear in role-name patterns
        }{
            {"km", 5000, false},
            {"kph", 5000, true},
            {"whereiskurt", 5000, true},  // 11-char prefix — upper bound case
        }
        for _, tc := range cases {
            t.Run(tc.prefix, func(t *testing.T) {
                got := BuildSCPPolicy(tc.prefix, "123456789012", "us-east-1")
                if len(got) > tc.maxBytes {
                    t.Errorf("policy size %d bytes exceeds threshold %d for prefix %q", len(got), tc.maxBytes, tc.prefix)
                }
                if tc.wantNoKM && strings.Contains(got, ":role/km-") {
                    t.Errorf("non-km prefix %q produced policy still containing km- role names: %s", tc.prefix, got)
                }
                expectedRoleSubstring := ":role/" + tc.prefix + "-ecs-spot-handler"
                if !strings.Contains(got, expectedRoleSubstring) {
                    t.Errorf("expected policy to contain %q, got: %s", expectedRoleSubstring, got)
                }
            })
        }
    }

    func TestBuildSCPPolicyPrefix(t *testing.T) {
        // Backward compat: BuildSCPPolicy("km", ...) renders identical role patterns to pre-84.4
        got := BuildSCPPolicy("km", "123456789012", "us-east-1")
        for _, want := range []string{
            "km-ecs-spot-handler",
            "km-budget-enforcer-*",
            "km-ec2spot-ssm-*",
            "km-github-token-refresher-*",
            "km-ttl-handler",
        } {
            if !strings.Contains(got, want) {
                t.Errorf("km-default policy missing expected role name %q", want)
            }
        }
    }
    ```
