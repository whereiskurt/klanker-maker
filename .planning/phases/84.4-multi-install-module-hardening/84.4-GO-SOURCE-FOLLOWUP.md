## Notes

- **Default install (`km`) is unaffected** by #2–#6 in practice because literal `km-` == the prefix. They bite only the second operator. #7 may be broken everywhere.
- **Hygiene-gate limitation:** the detector catches Go name-construction (concat / `fmt.Sprintf` / bare path-alias literals), NOT literals embedded inside HCL/template strings (#2). The `*.tf` module-hygiene test owns generated-HCL; #2 sits in the seam (Go-generated HCL) and needs a manual fix.
- Suggested sequencing: #6 + #3 (small, high-value, EC2/Docker metering+auth correctness) → #5 (doctor GC) → #2 (ECS) → #7 (investigate) → #4 (ECR, needs migration decision).
