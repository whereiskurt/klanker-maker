## FoundationStateReader Extension

No extension required. The existing `StateOwns(ctx context.Context, resourceAddr string) (bool, error)` interface handles DKIM/MX/TXT addresses by string matching — the same mechanism used for `aws_ses_receipt_rule_set.shared[0]` and `aws_ses_domain_identity.sandbox[0]`.

## Route53 Client Construction

The `r53ImportClient` is `route53.NewFromConfig(awsCfg)` where `awsCfg` uses the `klanker-terraform` AWS profile (same profile used for all other bootstrap API calls). Route53 is a global service, so the region in `awsCfg` is irrelevant for API calls. This mirrors the pattern in `unbootstrap.go`.
