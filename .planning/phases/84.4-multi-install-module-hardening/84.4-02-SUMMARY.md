## Decisions Made

- **Kept existing `BuildSCPPolicy` signature unchanged** — breaking it would break `bootstrap_test.go` (3 existing tests). Added `BuildSCPPolicyFromPrefix` as a new wrapper instead of renaming.
- **Minimal trustedBase (SSO only) in `BuildSCPPolicyFromPrefix`** — mirrors `var.trusted_role_arns` default in the HCL module. Using 6 ARNs in `trustedBase` caused the `whereiskurt` case to exceed 5000 bytes (5110 bytes). With SSO-only base, worst case is 3745 bytes — 1255 bytes of margin.
- **Did not add `lifecycle.prevent_destroy`** on the SCP resource — per plan instructions, it's per-install (not shared), and operators must be able to destroy via `km uninit` once that path covers SCP.
- **Added `km:resource-prefix` tag** to `aws_organizations_policy.sandbox_containment` for multi-install traceability (minor addition beyond plan spec, per Rule 2).
