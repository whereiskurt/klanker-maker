## Decisions Made

- Exported `GenerateClusterHCLWithOIDC` (capital G) rather than keeping it unexported, so the `cmd_test` (external) package can call it directly in the false-branch test without needing additional seams.
- `RunClusterAdd` receives `registerOIDCProviderFlag string` as the 11th parameter; the Cobra RunE closure passes `opts.registerOIDCProvider` directly. This keeps flag parsing + auto-detect inside `RunClusterAdd` while keeping the Cobra wiring clean.
- Tests pass `"false"` as the 11th argument to skip the IAM `ListOpenIDConnectProviders` call in unit tests that mock credentials via `NewClusterRunnerFunc` seam but do not mock the IAM layer.
- ARN-to-URL derivation uses `strings.SplitN(oidcProviderARN, ":oidc-provider/", 2)[1]` prefixed with `"https://"` — the simplest correct approach matching the Terraform `local.oidc_provider_url` pattern.

## Deviations from Plan
