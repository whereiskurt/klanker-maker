```go
type RuntimeDesktopSpec struct {
    Enabled  *bool    `json:"enabled,omitempty" yaml:"enabled,omitempty"`
    Mode     string   `json:"mode,omitempty" yaml:"mode,omitempty"`       // kiosk | full (default kiosk)
    Browsers []string `json:"browsers,omitempty" yaml:"browsers,omitempty"` // subset of {firefox,chromium,brave}, default [firefox]
    Geometry string   `json:"geometry,omitempty" yaml:"geometry,omitempty"` // default 1920x1080
}

func IsDesktopEnabled(desktop *RuntimeDesktopSpec) bool {
    if desktop == nil || desktop.Enabled == nil {
        return false
    }
    return *desktop.Enabled
}
```
Add to RuntimeSpec: `Desktop *RuntimeDesktopSpec \`yaml:"desktop,omitempty" json:"desktop,omitempty"\``

JSON Schema vscode block is at schemas/sandbox_profile.schema.json:273 — mirror it for desktop.
