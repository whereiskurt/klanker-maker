<objective>
Convert the four silent `km-…` literal fallbacks (F1 in CONTEXT.md `<decisions>`) into either hard failures (sidecar Lambdas) or prefix-aware helper calls (compiler template generation).

Today, missing `KM_BUDGET_TABLE` / `KM_SLACK_THREADS_TABLE` / `KM_SLACK_STREAM_TABLE` env vars cause silent fallback to `"km-…"` literals — a configuration bug that masquerades as success.

Output: configui + km-slack-bridge fail loudly on missing env; userdata.go template generator uses the existing `cfg.GetSlackThreadsTableName()` / `cfg.GetSlackStreamMessagesTableName()` helpers in place of three literal defaults.
</objective>
