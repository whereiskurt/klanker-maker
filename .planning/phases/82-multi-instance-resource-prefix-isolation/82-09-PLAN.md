<objective>
Tag every sandbox-creating Terraform resource with `km:resource-prefix = var.resource_prefix`, so Plan 04's doctor tag-filter and Plan 05's backfill operate against a consistent set of post-Phase-82 resources.

Purpose: Without these tag additions, every `km destroy && km create` cycle continues to produce untagged sandboxes that would WARN forever in Plan 04's doctor. The Terraform-side tag emission ensures NEW sandboxes are born tagged.

Output: Tag-map additions to ~6 module main.tf files plus `resource_prefix` variable declarations in the 3 ECS module variables.tf files (those are the only modules currently missing the variable).
</objective>
