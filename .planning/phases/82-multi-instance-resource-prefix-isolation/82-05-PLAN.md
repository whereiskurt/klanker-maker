<objective>
Ship the operator-facing `km doctor --backfill-tags` command that retrofits `km:resource-prefix=${prefix}` onto pre-Phase-82 resources, with a mandatory cross-install safety guard so a `kph` operator running this command cannot accidentally stamp their prefix onto an `rg` operator's resources.

Purpose: Without this command, every pre-Phase-82 resource (today's `km` install's existing infra) gets WARN-flagged forever by Plan 04's `checkOrphanedEC2`. The backfill closes that loop in one operator run.

Output: New file `internal/app/cmd/doctor_backfill_tags.go` implementing the command; cobra flag registration on the existing `km doctor` subcommand; two unit tests covering the cross-install guard and idempotency.
</objective>
