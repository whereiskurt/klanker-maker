<objective>
Close the configure-flow footgun (C1 in CONTEXT.md `<decisions>`). Three sites in `internal/app/cmd/configure.go` (lines 140, 178, 185) unconditionally reset `resourcePrefix` to `"km"` when the operator-supplied value is empty. An `rg` operator who re-runs `km configure` to rotate an unrelated field (e.g. operator_email) silently rewrites their config to point at the `km` install's tables.

Purpose: Make `km configure` safe to re-run for an existing non-default-prefix install. Preserve `resource_prefix` from disk by default; require an explicit `--reset-prefix` flag (or fresh-install path) to change it.

Output: Updated `configure.go` with preserve-on-re-run logic + new `--reset-prefix` flag; two new test cases in `configure_test.go` proving both behaviors.
</objective>
