<objective>
Extend the Phase 99 GitHub comment-verb parser to recognize reserved `/claude` and `/codex`
agent verbs as a separate composable axis from template commands, and carry the parsed verb
to the sandbox via a new `GitHubEnvelope.Agent` field.

Purpose: This is the bridge-side half of Phase 102 — without it, the poller (Plan 03) has no
`agent` field to read. It is pure-function Go, fully unit-testable, and lands in the same
`make build-lambdas` + `km init` deploy as the poller.

Output: `ParseResult` gains `AgentVerb`/`AgentVerbConflict`; `ParseCommands` intercepts the
reserved verbs; `ExtractArgs`/`RunCommandPass` strip the agent verb from `{{args}}`;
`GitHubEnvelope` gains `Agent`; `Handle()` posts the two-verb error reply and populates the
envelope.
</objective>
