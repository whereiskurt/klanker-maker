**recorder.go:**
- `NewRecorder() *Recorder` constructor
- `Recorder` struct with `sync.Mutex` protecting three `map[string]struct{}` fields: `dnsObserved`, `hostObserved`, `reposObserved`
- `RecordDNSQuery(domain string)` — `strings.TrimSuffix(domain, ".")`, `strings.ToLower`, store in `dnsObserved`
- `RecordHost(hostPort string)` — use `net.SplitHostPort` (fall back to raw string if no port), lowercase, store in `hostObserved`
- `RecordRepo(ownerRepo string)` — lowercase, store in `reposObserved`
- `HandleTLSEvent(event *tls.TLSEvent) error` — satisfies `tls.EventHandler` signature. Only process `event.Direction == 1` (DirWrite). Call `tls.ParseHTTPRequest(event.PayloadBytes())`, on error return nil (skip). Record `req.Host` via `RecordHost`. Call `tls.ExtractGitHubRepo(req.Host, req.Path)`, if owner non-empty record `owner + "/" + repo` via `RecordRepo`.
- Accessor methods: `DNSDomains() []string`, `Hosts() []string`, `Repos() []string` — return sorted slices from the maps (for generator and testing)

**normalize.go:**
- `CollapseToDNSSuffixes(domains []string) []string` — for each domain, extract eTLD+1 using `golang.org/x/net/publicsuffix.EffectiveTLDPlusOne()`. Prefix with ".". Collect unique suffixes, sort, return.
- Handle error from `EffectiveTLDPlusOne` (bare TLD like "com") by skipping that entry.
- This avoids over-permissive suffixes because `EffectiveTLDPlusOne("com")` returns an error.

**generator.go:**
- `func (r *Recorder) Generate(base string) (*profile.SandboxProfile, error)` — lock mutex, call `CollapseToDNSSuffixes(r.DNSDomains())` for suffixes, call `deduplicateHosts(r.Hosts(), suffixes)` for hosts, get `r.Repos()` for repos. Build `profile.SandboxProfile` with:
  - `APIVersion: "klankermaker.ai/v1alpha1"`, `Kind: "SandboxProfile"`
  - `Metadata.Name: "observed-" + time.Now().Format("20060102-150405")`
  - `Extends: base` (if non-empty)
  - `Spec.Network.Egress.AllowedDNSSuffixes: suffixes`
  - `Spec.Network.Egress.AllowedHosts: hosts`
  - `Spec.SourceAccess.GitHub.AllowedRepos: repos` (only if repos non-empty)
  - Always use empty slices not nil (so YAML emits `[]` not `null`)

- `func deduplicateHosts(hosts []string, suffixes []string) []string` — for each host, check if any suffix matches (host ends with suffix without leading dot, or "." + host ends with suffix). If covered, omit. Return sorted remainder.

- `func (r *Recorder) GenerateYAML(base string) ([]byte, error)` — calls Generate, then `goccy/go-yaml.Marshal`. Add a comment header: `# Generated by km observe -- review before use`.
