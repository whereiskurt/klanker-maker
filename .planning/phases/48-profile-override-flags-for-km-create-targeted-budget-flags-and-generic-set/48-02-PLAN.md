<objective>
Wire TTL=0 idle-hibernate loop: compiler passes IDLE_ACTION=hibernate to the audit-log sidecar via userdata, and the sidecar loops (hibernate on idle, re-arm detector) instead of exiting.

Purpose: When --ttl 0 is used, the sandbox should never self-destroy. Instead, each idle interval triggers hibernate/stop, and when the sandbox resumes, idle detection re-arms automatically. This creates an indefinite hibernate-on-idle loop.

Output: Modified compiler userdata with IdleAction field, create.go sets it when TTL=0, sidecar reads IDLE_ACTION and loops when "hibernate". Tests for both compiler and sidecar paths.
</objective>

       ```go
       var startIdleDetector func()
       startIdleDetector = func() {
           detector := newIdleDetector(sandboxID, idleMinutes, cwClient, cwLogGroup, "audit", func(id string) {
               if idleAction == "hibernate" {
                   log.Warn().Str("sandbox_id", id).Msg("audit-log: sandbox idle, sending hibernate (TTL=0 loop)")
                   if ebClient != nil {
                       if err := kmaws.PublishSandboxCommand(ctx, ebClient, id, "stop"); err != nil {
                           log.Error().Err(err).Str("sandbox_id", id).Msg("audit-log: failed to publish stop command")
                       }
                   }
                   // Re-arm: the EC2 instance will hibernate. When it resumes, the sidecar
                   // process is still alive (hibernate preserves RAM). After a brief delay
                   // to allow the instance to fully hibernate and resume, start a new detector.
                   // The new detector's startTime resets, so it waits for genuinely new activity.
                   time.AfterFunc(2*time.Minute, startIdleDetector)
               } else {
                   log.Warn().Str("sandbox_id", id).Msg("audit-log: sandbox idle timeout reached, publishing idle event")
                   if ebClient != nil {
                       if err := kmaws.PublishSandboxIdleEvent(ctx, ebClient, id); err != nil {
                           log.Error().Err(err).Str("sandbox_id", id).Msg("audit-log: failed to publish idle event")
                       }
                   }
                   cancel()
               }
           })
           go func() {
               if runErr := detector.Run(ctx); runErr != nil && runErr != context.Canceled {
                   log.Error().Err(runErr).Msg("audit-log: idle detector error")
               }
           }()
           log.Info().Int("idle_timeout_minutes", idleMinutes).Str("idle_action", idleAction).Msg("audit-log: idle detector started")
       }
       startIdleDetector()
       ```
