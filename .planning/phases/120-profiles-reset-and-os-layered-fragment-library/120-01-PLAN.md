Existing fragment shape (all set metadata.abstract: true, only declare fields they OWN):
```yaml
apiVersion: klankermaker.ai/v1alpha2
kind: SandboxProfile
metadata:
  name: base-<slug>
  abstract: true

  <only the block(s) this fragment owns>
```

deepMerge / extends mechanics (from pkg/profile/inherit.go, verified):
- extends order is left→right; child merged LAST; scalar collision → child/right-parent wins.
- list fields (initCommands, allowedDNSSuffixes, autoApprove, rsyncPaths, configFiles keys,
  invites.emails, …) CONCAT + DEDUP (concatDedup, reflect.DeepEqual, first-occurrence kept).
- initCommands union is concat-in-order, first-wins → a leaf that lists base/os/* FIRST gets
  OS lines before toolchain lines.
- metadata.abstract:true ⇒ km validate returns SKIP + exit 0 (IsAbstractFragment).

BOOL ZERO-VALUE TRAP (load-bearing): a fragment that writes a non-pointer bool (spot,
hibernation, mountEFS) forces its zero-value onto every child that omits the field. OS
fragments MUST declare ONLY the string `spec.runtime.ami` under spec.runtime — nothing else.
