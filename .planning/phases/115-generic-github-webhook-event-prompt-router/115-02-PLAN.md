  type EventRule struct {
      On              string   `json:"on"`
      Actions         []string `json:"actions,omitempty"`
      Match           string   `json:"match"`
      Exclude         []string `json:"exclude,omitempty"`
      Profile         string   `json:"profile,omitempty"`
      Alias           string   `json:"alias,omitempty"`
      Agent           string   `json:"agent,omitempty"`
      CooldownSeconds int      `json:"cooldown_seconds,omitempty"`
      Prompt          string   `json:"prompt"`
  }
  type EventPayload struct { Repo, Action, Sender, DefaultBranch, HTMLURL string }

  func MatchEventRule(eventType string, payload EventPayload, rules []EventRule) *EventRule
    Two-pass, mirroring Resolve (resolve.go:51-70):
      helper ruleMatches(r EventRule) bool := r.On==eventType
          && (len(r.Actions)==0 || contains(r.Actions, payload.Action))
          && !excluded(r.Exclude, payload.Repo)
      Pass 1 (exact): for each rule where ruleMatches && r.Match==payload.Repo → return &rule (copy via local var to avoid loop-var aliasing).
      Pass 2 (glob): for each rule where ruleMatches && isGlob(r.Match) && path.Match(r.Match, payload.Repo)==(true,nil) → return &rule.
      else return nil.
      excluded(globs, repo): true if any path.Match(g, repo) is true (and for exact non-glob exclude entries, g==repo). Ignore path.Match errors (treat as no-match), consistent with resolve.go.
    REUSE isGlob from resolve.go (do not redefine). Import "path".

  func ExpandEventTemplate(tmpl string, p EventPayload, eventType string) string
    Use strings.NewReplacer with the six pairs (see RESEARCH.md Pattern 3) and return r.Replace(tmpl).
    Same no-text/template decision as ExpandTemplate (commands.go:357).

Then edit pkg/github/bridge/payload.go: add `HTMLURL string \`json:"html_url"\`` to RepositoryField
(payload.go:54) so {{html_url}} can be sourced from repository.html_url for repository/push events.
