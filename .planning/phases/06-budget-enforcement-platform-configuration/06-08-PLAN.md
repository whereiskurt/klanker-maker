<objective>
Wire spot rate resolution into the sandbox creation flow so compute budget enforcement works at runtime.

Purpose: BUDG-03 requires compute spend tracked as spot rate x elapsed minutes. The Lambda enforcer calculates correctly, but the compiler hardcodes SpotRateUSD=0.0 — so compute spend is always $0.00 and enforcement never triggers. This gap closure wires GetSpotRate into create.go and passes the resolved rate through to the compiler.

Output: Real spot rates flow from Pricing API (or static fallback) through create.go -> NetworkConfig -> compiler -> service.hcl -> Lambda, enabling non-zero compute spend tracking.
</objective>

    if resolvedProfile.Spec.Budget != nil && resolvedProfile.Spec.Budget.Compute != nil {
        instanceType := resolvedProfile.Spec.Runtime.InstanceType
        if instanceType == "" {
            instanceType = "t3.medium" // default instance type

        // Try Pricing API first, fall back to static table
        pricingCfg := awsCfg.Copy()

            spotRate = staticSpotRate(instanceType)
            log.Warn().Err(err).Str("instanceType", instanceType).
