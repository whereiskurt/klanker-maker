**Event payload** (from EventBridge Scheduler, same as TTL handler):
```go
type BudgetCheckEvent struct {
    SandboxID    string `json:"sandbox_id"`
    InstanceType string `json:"instance_type"`   // for spot rate lookup
    SpotRate     float64 `json:"spot_rate"`       // pre-calculated hourly rate (set at creation)
    Substrate    string `json:"substrate"`        // "ec2" or "ecs"
    CreatedAt    string `json:"created_at"`       // RFC3339 timestamp
    RoleARN      string `json:"role_arn"`         // IAM role to revoke Bedrock from
    InstanceID   string `json:"instance_id"`      // EC2 instance ID (empty for ECS)
    TaskARN      string `json:"task_arn"`         // ECS task ARN (empty for EC2)
    OperatorEmail string `json:"operator_email"`  // for notifications
}
```

**Handler logic:**
1. Calculate elapsed minutes since CreatedAt
2. Compute cost = spotRate * (elapsedMinutes / 60)
3. Call IncrementComputeSpend (this is idempotent via SET, not ADD — use SET with the absolute computed value since Lambda re-calculates from scratch each minute)
   - Actually, use UpdateItem SET spentUSD = :cost (not ADD) since compute spend is recalculated each invocation, not incremental
4. Call GetBudget to read current state of both pools
5. Check thresholds:
   - **80% compute OR 80% AI**: Send warning email via SendLifecycleNotification (only once — check a `warningNotified` DynamoDB attribute, set it after first notification)
   - **100% compute**: Suspend sandbox. EC2: ec2.StopInstances. ECS: trigger artifact upload (via S3 stored profile), then ecs.StopTask. Send enforcement email.
   - **100% AI (backstop)**: Detach Bedrock IAM policy from role. iam.DetachRolePolicy with the Bedrock access policy ARN. This is a backstop for SDK calls that bypass the proxy. Send enforcement email.
6. All AWS operations use narrow interfaces for testability

**Narrow interfaces:**
```go
type EC2StopAPI interface {
    StopInstances(ctx, *ec2.StopInstancesInput, ...opts) (*ec2.StopInstancesOutput, error)
}
type IAMDetachAPI interface {
    DetachRolePolicy(ctx, *iam.DetachRolePolicyInput, ...opts) (*iam.DetachRolePolicyOutput, error)
}
type ECSStopAPI interface {
    StopTask(ctx, *ecs.StopTaskInput, ...opts) (*ecs.StopTaskOutput, error)
}
```

**Env vars:** KM_BUDGET_TABLE, KM_EMAIL_DOMAIN, KM_STATE_BUCKET (for reading stored profile on ECS teardown).

Tests use mock implementations for all interfaces. Follow the exact same mock pattern as cmd/ttl-handler/main_test.go.
