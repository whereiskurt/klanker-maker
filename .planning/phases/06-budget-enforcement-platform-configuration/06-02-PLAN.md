Create pkg/aws/budget.go following the narrow interface pattern:

```go
type BudgetAPI interface {
    UpdateItem(ctx, *dynamodb.UpdateItemInput, ...opts) (*dynamodb.UpdateItemOutput, error)
    GetItem(ctx, *dynamodb.GetItemInput, ...opts) (*dynamodb.GetItemOutput, error)
    Query(ctx, *dynamodb.QueryInput, ...opts) (*dynamodb.QueryOutput, error)
}
```

Implement helper functions (all take BudgetAPI + tableName):
- IncrementAISpend(ctx, client, tableName, sandboxID, modelID string, inputTokens, outputTokens int, costUSD float64) — uses UpdateItem ADD for atomic increment, returns updated spend total
- IncrementComputeSpend(ctx, client, tableName, sandboxID string, costUSD float64) — atomic increment on BUDGET#compute
- GetBudget(ctx, client, tableName, sandboxID string) (*BudgetSummary, error) — Query PK=SANDBOX#{id} SK begins_with BUDGET#, returns structured summary with compute/AI/limits
- SetBudgetLimits(ctx, client, tableName, sandboxID string, computeLimit, aiLimit, warningThreshold float64) — PutItem or UpdateItem on BUDGET#limits

BudgetSummary struct:
```go
type BudgetSummary struct {
    ComputeSpent  float64
    ComputeLimit  float64
    AISpent       float64 // total across models
    AILimit       float64
    WarningThreshold float64
    AIByModel     map[string]ModelSpend // keyed by model ID
}
type ModelSpend struct {
    SpentUSD     float64
    InputTokens  int
    OutputTokens int
}
```

Create pkg/aws/pricing.go:
- PricingAPI interface wrapping pricing.GetProducts
- GetSpotRate(ctx, client, instanceType, region string) (float64, error) — query EC2 spot pricing, return hourly rate
- GetBedrockModelRates(ctx, client PricingAPI) (map[string]BedrockModelRate, error) — query AmazonBedrock pricing, with static fallback table
- BedrockModelRate struct: ModelID, InputPricePer1KTokens, OutputPricePer1KTokens float64
