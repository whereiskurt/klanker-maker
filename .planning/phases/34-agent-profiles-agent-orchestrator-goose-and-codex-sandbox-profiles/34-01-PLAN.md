**profiles/agent-orchestrator.yaml** (ComposioHQ multi-agent coordinator):
- metadata: name=agent-orchestrator, labels={tier: development, tool: agent-orchestrator, builtin: "true"}, prefix=ao, alias=ao
- lifecycle: ttl=8h, idleTimeout=1h, teardownPolicy=destroy (longer TTL for multi-agent sessions)
- runtime: substrate=ec2, spot=true, instanceType=t3.large, region=us-east-1 (needs more memory for tmux sessions + multiple agents)
- execution.env: SANDBOX_MODE=agent-orchestrator, GITHUB_TOKEN="" (placeholder — injected from SSM at provision time; required for gh auth)
- execution.rsyncPaths: .gitconfig, .agent-orchestrator
- execution.initCommands:
  1. `yum install -y git nodejs npm tmux jq` (core deps)
  2. `yum install -y 'dnf-command(config-manager)' && dnf config-manager --add-repo https://cli.github.com/packages/rpm/gh-cli.repo && dnf install -y gh` (GitHub CLI — required for ao PR/issue workflows)
  3. `npm install -g @composio/ao` (agent-orchestrator CLI)
  4. `echo "$GITHUB_TOKEN" | gh auth login --with-token` (authenticate gh using token injected from SSM — ao start will fail without this)
  5. `mkdir -p /workspace ~/.agent-orchestrator ~/.worktrees && chown -R sandbox:sandbox /workspace ~/.agent-orchestrator ~/.worktrees`
  6. `su - sandbox -c 'cd /workspace && git config --global user.name "Agent Orchestrator Sandbox" && git config --global user.email "sandbox@klankermaker.ai"'`
- sourceAccess: same allowlist pattern as claude-dev.yaml (allowedRepos, allowedRefs, permissions)
- network.egress.allowedDNSSuffixes: .amazonaws.com, .npmjs.org, .npmjs.com, .nodejs.org, .github.com, .githubusercontent.com
- network.egress.allowedHosts: registry.npmjs.org, api.github.com, github.com
- network.egress.allowedMethods: GET, POST, PUT, PATCH, DELETE
- budget: compute.maxSpendUSD=4.00, ai.maxSpendUSD=10.00, warningThreshold=0.80 (higher budgets for multi-agent)
- artifacts: paths=[/workspace], maxSizeMB=1000
- identity: roleSessionDuration=1h, allowedRegions=[us-east-1], sessionPolicy=minimal
- sidecars: ALL enabled (dnsProxy, httpProxy, auditLog, tracing) with km-* images
- observability: commandLog + networkLog to cloudwatch, claudeTelemetry enabled
- policy: allowShellEscape=false, allowedCommands=[git, node, npm, npx, ao, tmux, gh, bash, sh, jq]
- filesystemPolicy: readOnlyPaths=[], writablePaths=[/workspace, /tmp, /home]

- agent: maxConcurrentTasks=4, taskTimeout=120m, allowedTools=[bash, read_file, write_file, list_files]
