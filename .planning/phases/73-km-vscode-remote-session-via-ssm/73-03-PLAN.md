<algorithm>
**Parse phase (split file into 3 regions):**
1. Read entire file (or treat as empty if `os.IsNotExist`).
2. Walk lines top-to-bottom. Find the FIRST occurrence of `beginMarker` and the FIRST
   subsequent occurrence of `endMarker`.
3. Region split:
   - `before`: bytes from offset 0 up to (but not including) the line containing beginMarker.
     If beginMarker is absent, before = entire file content.
   - `inside`: bytes from the line AFTER beginMarker up to (but not including) the line
     containing endMarker. Empty if no markers.
   - `after`: bytes from the line AFTER endMarker to EOF. Empty if no markers.
4. Within `inside`, split into Host blocks: a block starts at a line matching `^Host\s+(\S+)\s*$`
   and runs until the next `Host` line OR the end of `inside`.

**Upsert phase:**
1. Build the new Host block string:
   ```
   Host <alias>
     HostName <opts.HostName>
     Port <opts.Port>
     User <opts.User>
     IdentityFile <opts.IdentityFile>
     IdentitiesOnly yes
     StrictHostKeyChecking no
     UserKnownHostsFile /dev/null
     ServerAliveInterval 30
   ```
   (followed by a newline).
2. Iterate parsed Host blocks; if one matches alias, replace it; else append the new block.
3. Reassemble: `before + beginMarker + "\n" + (joined Host blocks) + endMarker + "\n" + after`.
4. If `before` was the entire file (no markers existed) and `before` is non-empty and doesn't
   end with `"\n"`, insert a `"\n"` between before and beginMarker.

**Remove phase:**
1. Iterate parsed Host blocks; drop the one matching alias.
2. If zero Host blocks remain after removal: assemble as `before + after`. If `before` is
   non-empty and `after` is non-empty and `before` doesn't end with `"\n"`, insert one.
3. Else: reassemble as `before + beginMarker + "\n" + (joined remaining blocks) + endMarker + "\n" + after`.

**Atomic write:**
1. `tempFile, _ := os.CreateTemp(filepath.Dir(configPath), ".km-ssh-config-*")`.
2. Write content + close + chmod 0600.
3. `os.Rename(tempFile, configPath)` — atomic on macOS/Linux.

**File-doesn't-exist case for UpsertHost:**
- Write the assembled content with `os.WriteFile(configPath, [content], 0600)`. The atomic
  temp+rename dance is for the modify-existing case; for create-from-scratch the WriteFile
  is fine.

**File-doesn't-exist case for RemoveHost:**
- Return nil immediately (idempotent).
</algorithm>

    func atomicWrite(path string, content []byte, mode os.FileMode) error {
        if err := os.MkdirAll(filepath.Dir(path), 0o700); err != nil {
            return fmt.Errorf("sshconfig: mkdir %s: %w", filepath.Dir(path), err)
        }
        tmp, err := os.CreateTemp(filepath.Dir(path), ".km-ssh-config-*")
        if err != nil {
            return fmt.Errorf("sshconfig: create temp: %w", err)
        }
        tmpPath := tmp.Name()
        defer os.Remove(tmpPath) // no-op on success after Rename
        if _, err := tmp.Write(content); err != nil {
            tmp.Close()
            return fmt.Errorf("sshconfig: write temp: %w", err)
        }
        if err := tmp.Close(); err != nil {
            return fmt.Errorf("sshconfig: close temp: %w", err)
        }
        if err := os.Chmod(tmpPath, mode); err != nil {
            return fmt.Errorf("sshconfig: chmod temp: %w", err)
        }
        if err := os.Rename(tmpPath, path); err != nil {
            return fmt.Errorf("sshconfig: rename temp -> %s: %w", path, err)
        }
        return nil
    }

    ```go
    func RemoveHost(configPath, alias string) error {
        content, err := os.ReadFile(configPath)
        if err != nil {
            if os.IsNotExist(err) {
                return nil
            }
            return fmt.Errorf("sshconfig: read %s: %w", configPath, err)
        }
        before, inside, after := managedSections(content)
        if inside == nil {
            // No markers — nothing to remove.
            return nil
        }
        blocks := parseHostBlocks(inside)
        kept := blocks[:0]
        for _, b := range blocks {
            if b.alias != alias {
                kept = append(kept, b)
            }
        }
        if len(kept) == len(blocks) {
            // Alias not present — idempotent no-op.
            return nil
        }

        var out bytes.Buffer
        out.Write(before)
        ensureTrailingNewline(&out)

        if len(kept) == 0 {
            // Drop markers entirely; just join before + after.
            out.Write(after)
        } else {
            out.WriteString(beginMarker + "\n")
            for _, b := range kept {
                out.Write(b.content)
            }
            out.WriteString(endMarker + "\n")
            out.Write(after)
        }
        return atomicWrite(configPath, out.Bytes(), 0o600)
    }

    func ensureTrailingNewline(buf *bytes.Buffer) {
        if buf.Len() == 0 {
            return
        }
        if buf.Bytes()[buf.Len()-1] != '\n' {
            buf.WriteByte('\n')
        }
    }
    ```

    Edge: when `len(kept) == 0` AND `before` is empty AND `after` is empty, `out` will be empty
    after the writes. Atomic-write an empty file (mode 0600) — this is correct behavior. The
    operator's file becomes empty, which is fine because their original ~/.ssh/config consisted
    SOLELY of our managed block. Document this in a comment.
