1. Create `cmd/create-handler/main.go` mirroring `cmd/ttl-handler/main.go` structure:

   - Define `CreateHandler` struct with fields: `S3Client *s3.Client`, `SESClient awspkg.SESV2API`, `Domain string`, `KMBinaryPath string` (default `/var/task/km`)
   - Implement `Handle(ctx context.Context, event CreateEvent) error`:
     a. Download `.km-profile.yaml` from S3 at `{artifact_prefix}.km-profile.yaml`
     b. Write profile to `/tmp/{sandbox-id}.yaml`
     c. Build `exec.CommandContext` for km binary: `[kmBinaryPath, "create", profilePath, "--aws-profile", ""]`
     d. If `event.OnDemand`, append `"--on-demand"` arg
     e. Set env vars: `KM_ARTIFACTS_BUCKET`, `KM_OPERATOR_EMAIL`, `KM_REMOTE_CREATE=true` (so create.go can detect Lambda context if needed)

     d. Create `CreateHandler` and call `lambdaruntime.Start(handler.Handle)`
