<objective>
Extend the http-proxy sidecar to meter Anthropic direct API calls (api.anthropic.com/v1/messages) for AI budget enforcement. Sandboxes running Claude Code against the Anthropic 1P endpoint get the same per-token budget tracking, threshold warnings, and hard enforcement (proxy 403) as Bedrock workloads.

Purpose: Claude Code uses api.anthropic.com by default (not Bedrock). Without this, AI spend from Claude Code sessions is untracked and unenforceable.
Output: anthropic.go + anthropic_test.go + proxy.go modifications. Full TDD cycle.
</objective>
