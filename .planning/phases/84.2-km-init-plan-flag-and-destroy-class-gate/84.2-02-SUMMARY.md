## What Was Built

### pkg/terragrunt/planreport/protected.go
Defines `var ProtectedTypes = []string{...}` with exactly 9 entries, each annotated with the incident that motivated it (Phase 84 Gap 3, Phase 84 Gap 6, etc.). Adding to the list requires PR review — no operator-side config file exists by design (CONTEXT.md Decision 6).

### pkg/terragrunt/planreport/report.go
`Parse(jsonBytes []byte) (Report, error)` — hand-rolls a minimal struct (`rawPlan`, `rawResourceChange`, `rawChange`) over `encoding/json`. Only 3 fields per resource_change consumed (address, type, change.actions). Accepts any `format_version` starting with `"1."`, rejects anything else. `classifyAction` maps Terraform's actions array to create/update/delete/replace/no-op, handling both `["delete","create"]` and `["create","delete"]` orderings.

### pkg/terragrunt/planreport/gate.go
`Evaluate(reports []Report, acceptDestroys bool) GateResult` implementing the locked gate algorithm from CONTEXT.md decisions lines 117-135. ParseFailed reports are conservative-tripped (`PARSE-FAIL`). `acceptDestroys=true` clears `Blocked` but `Trips` is still populated (operator visibility contract).
