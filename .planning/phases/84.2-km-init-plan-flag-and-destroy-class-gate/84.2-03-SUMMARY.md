## What Was Built

Two new Runner methods and two new Build* helpers added to `pkg/terragrunt/runner.go`:

- `BuildPlanWithOutputCommand(ctx, sandboxDir, planFile)` — constructs `terragrunt plan -out=<planFile>` exec.Cmd
- `BuildShowPlanJSONCommand(ctx, sandboxDir, planFile)` — constructs `terragrunt show -json <planFile>` exec.Cmd  
- `PlanWithOutput(ctx, sandboxDir, planFile, stdoutBuf)` — runs plan with stdout captured into caller-supplied buffer, inherits Phase 84.1-02 runBounded ctx bounding + heartbeat
- `ShowPlanJSON(ctx, sandboxDir, planFile)` — runs show -json and returns clean stdout bytes via cmd.Output(), suitable for planreport.Parse
