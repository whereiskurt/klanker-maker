    ```go
    // checkStaleAMIs lists km-tagged self-owned AMIs in the given region and
    // flags those that are (a) older than staleDays AND (b) not referenced by any
    // profile in profileSearchPaths AND (c) not backing any running sandbox.
    // Returns CheckOK when no stale AMIs are found, CheckWarn with the list otherwise.
    // No deletion is performed in Phase 56 (flag-only).
    func checkStaleAMIs(ctx context.Context, region string, amiClient kmaws.EC2AMIAPI, lister SandboxLister, profileSearchPaths []string, staleDays int) CheckResult
    ```

    Logic:
    1. If amiClient is nil: return `CheckSkipped` with "EC2 AMI client not available".
    2. Call `kmaws.ListBakedAMIs(ctx, amiClient)`. On error: return `CheckWarn` with "could not list AMIs: %v".
    3. Compute threshold: `time.Now().UTC().Add(-time.Duration(staleDays) * 24 * time.Hour)`.
    4. For each Image:
       - Parse `Image.CreationDate` (format `2006-01-02T15:04:05.000Z`); skip if unparsable.
       - If creation time >= threshold: SKIP (not stale by age).
       - Build amiID = `*Image.ImageId`.
       - Check `FindProfilesReferencingAMI(profileSearchPaths, amiID)`. If non-empty: SKIP (referenced by profile).
       - Check running sandboxes via `lister.ListSandboxes(ctx, false)`. For each sandbox, parse its profile (resolved via cfg.ProfileSearchPaths) and check `Spec.Runtime.AMI == amiID`. If any match: SKIP. (Tolerate parse errors per-sandbox; log and continue.)
       - Add amiID + age to the stale list.
    5. If stale list is empty: return `CheckOK` with name "Stale AMIs (<region>)" and message "No stale AMIs in <region> (threshold: <N>d)".
    6. Otherwise: return `CheckWarn` with name "Stale AMIs (<region>)" and message listing each stale AMI ID + age + creation date. Format: `"3 stale AMIs in us-east-1: ami-0abc (45d), ami-0def (60d), ami-0ghi (90d)"`.
