<objective>
Implement the `km list`, `km status`, and `km logs` CLI commands — the operational visibility layer for running sandboxes. These commands let operators see what is running without opening the AWS console.

Purpose: Implements PROV-03 (km list) and PROV-04 (km status). km logs is the operational companion needed for the audit log sidecar (OBSV-03) to be useful. All three follow the established `NewXxxCmd(cfg *config.Config)` Cobra pattern. km list reads sandbox metadata.json written by km create (Plan 03-04) so profile, substrate, and TTL fields are populated.

Output: Three new Cobra command files + tests registered in root.go.
</objective>

- Human-readable table: sandbox ID, profile, substrate, region, status, TTL remaining
- --json: JSON array of sandbox objects
- --tags: use AWS tag-based scan instead of S3 prefix scan
- Diff between S3 and tags scan detects orphans

km status output (from 03-CONTEXT.md — LOCKED):
- Detailed: resources (IDs/ARNs), metadata (profile name, substrate), timestamps (created, TTL expiry)
- No inline audit log — that is km logs

km logs (from 03-CONTEXT.md — LOCKED):
- CloudWatch log group: /km/sandboxes/<sandbox-id>/
- Log stream: "audit"
- --follow: tail continuously until Ctrl+C (ctx cancellation)

S3 state scan: list objects under "tf-km/sandboxes/" prefix in the state bucket. Each sandbox has a directory entry. Parse sandbox-id from the key.

    // TestLogsCmd_ConstructsCorrectLogGroup verifies that the logs command builds the

    func TestLogsCmd_ConstructsCorrectLogGroup(t *testing.T) {
        sandboxID := "sb-abc123"
        expected := "/km/sandboxes/" + sandboxID + "/"

        // Construct the log group path using the same logic as logs.go RunE.
        // This is a pure string construction test — no AWS calls needed.
        logGroup := "/km/sandboxes/" + sandboxID + "/"

        if !strings.HasPrefix(logGroup, "/km/sandboxes/") {
            t.Errorf("log group %q does not start with /km/sandboxes/", logGroup)
        }
        if logGroup != expected {
            t.Errorf("log group = %q, want %q", logGroup, expected)
        }
        if !strings.HasSuffix(logGroup, "/") {

        }
    }
