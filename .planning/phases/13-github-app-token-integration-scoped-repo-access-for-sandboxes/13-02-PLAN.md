  <files>infra/modules/github-token/v1.0.0/main.tf, infra/modules/github-token/v1.0.0/variables.tf, infra/modules/github-token/v1.0.0/outputs.tf</files>
  <action>
    Create `infra/modules/github-token/v1.0.0/` mirroring the budget-enforcer module structure.

    **variables.tf:** Define variables: `sandbox_id` (string), `lambda_zip_path` (string), `installation_id` (string), `ssm_parameter_name` (string, default "/sandbox/{sandbox_id}/github-token"), `allowed_repos` (list(string)), `permissions` (string, JSON-encoded permissions map), `sandbox_iam_role_arn` (string, for KMS decrypt grant to sandbox).

    **main.tf:** Create resources:
    1. `aws_kms_key.github_token` — encryption for SSM SecureString, enable_key_rotation=true, deletion_window_in_days=7. Key policy: root admin + Lambda role encrypt + sandbox IAM role decrypt.
    2. `aws_kms_alias.github_token` — alias `alias/km-github-token-${var.sandbox_id}`
    3. `aws_iam_role.github_token_refresher` — name `km-github-token-refresher-${var.sandbox_id}`, assume Lambda service. Tags: `km:component=github-token-refresher`, `km:sandbox_id`.
    4. `aws_iam_role_policy.github_token_refresher` — inline policy granting: ssm:GetParameter on `/km/config/github/*`, ssm:PutParameter on the sandbox token path, kms:Encrypt+kms:Decrypt on the github_token KMS key, logs:CreateLogGroup+CreateLogStream+PutLogEvents.
    5. `aws_lambda_function.github_token_refresher` — function_name `km-github-token-refresher-${var.sandbox_id}`, runtime `provided.al2023`, handler `bootstrap`, architectures `["arm64"]`, timeout 60, memory_size 128. Environment: `KM_GITHUB_SSM_CONFIG_PREFIX=/km/config/github`.
    6. `aws_iam_role.scheduler_invoke` — assume scheduler.amazonaws.com, inline policy to invoke the Lambda.
    7. `aws_scheduler_schedule.github_token_refresh` — name `km-github-token-${var.sandbox_id}`, schedule_expression `rate(45 minutes)`, flexible_time_window OFF. Target: Lambda ARN with input payload containing sandbox_id, installation_id, ssm_parameter_name, kms_key_arn, allowed_repos, permissions. retry_policy max_retry_attempts=0.
    8. Tag ALL resources with `km:sandbox_id = var.sandbox_id`.

    **outputs.tf:** Output lambda_function_arn, kms_key_arn, kms_key_id.
  </action>
