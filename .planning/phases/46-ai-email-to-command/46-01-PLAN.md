   - Marshal haikuRequest with anthropic_version "bedrock-2023-05-31", max_tokens 512, temperature 0.1
   - Call client.InvokeModel with ModelId, ContentType "application/json", Accept "application/json"
   - Unmarshal response body into haikuResponse
   - Extract content[0].text, pass to parseHaikuResponse
   - Return parsed InterpretedCommand

   - First unmarshal into a map with json.Number for lenient type handling

   - Validate confidence in [0, 1] range
   - Default overrides to empty map if nil
   - Return InterpretedCommand

   - Specify the required JSON output format: {"command", "profile", "overrides", "confidence", "reasoning"}
   - Instruct Haiku to set confidence < 0.7 when ambiguous

type ConversationState struct {
    ThreadID     string            `json:"thread_id"`
